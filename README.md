# Turtle Graphics on Processing

Processingでタートルグラフィックス

部活の1年生の夏休みの課題にしようと考え，作成した．
タートルグラフィックスをProcessing上で*実装する*課題で，自分たちのプログラミングスキルを伸ばすと共に，中学生や小学生にもプログラミングを体験してもらえる教材が完成する．


## 課題の内容（考え中）

このテンプレートを1年生に配布し，メソッドをどんどん実装する形式の課題にしようと考えている．
例えば以下の様なメソッドなどを課題内容にする．

* カメを動かすメソッド
* ペンの操作をするメソッド
* 色を変えるメソッド
* 裏コマンド？
* カメの絵柄を変えるメソッド
* カメをアニメーションで動かすメソッド
* などなど...

遊び心があって，小学生や中学生が楽しめるような機能をたくさん実装してみて欲しい．

## 発展課題

オブジェクト指向入門として，複数のカメを登場させるために「クラス」を使う．
クラスを使ってタートルグラフィックスを実装するのはとても分かりやすい例かも...
